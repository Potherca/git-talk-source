* Requires a GPG key

Signing your work

Use an existing GPG key:

gpg --list-secret-keys --keyid-format LONG

or, generate a new GPG key:

gpg --full-generate-key

or, for GPG version lower than 2.1.17

gpg --default-new-key-algo rsa4096 --gen-key

Specify the Key


[user]
        signingkey 0A1B2C3D

user.signingkey 0A1B2C3D

https://git-scm.com/docs/git-config#git-config-usersigningKey

Always sign commits


[commit]
        gpgSign = true

commit.gpgSign true

https://git-scm.com/docs/git-config#git-config-commitgpgSign
